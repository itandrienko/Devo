# Devo version 1.0

React / TypeScript 

## Install

Clone the project

```bash
  git clone https://github.com/itandrienko/Devo.git
```

Install dependencies

```bash
  npm install
```

Build the server

```bash
  npm run build
```
## Run

```bash
  npm run start
```
